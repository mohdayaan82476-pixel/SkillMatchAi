🔥 AI-Powered Job Matching + Interview Platform | Built for Hackathons 🚀
SkillMatch AI is an intelligent career assistant platform that helps users match their skills to real job opportunities, improve their resumes, and prepare for interviews using AI.

---

## ✨ Features

### 🤖 AI Resume Analysis (ATS Scanner)
- Upload resume and get ATS compatibility score
- Shows matched skills and missing keywords
- Provides actionable improvement suggestions

### 💼 Real Job Matching (Live Data)
- Fetches real jobs using JSearch API
- Matches jobs based on your skills
- AI-powered match score (realistic scoring system)

### 🎯 Smart Job Insights
- Skill gap detection
- Salary insights (real market ranges)
- Best time to apply suggestions
- Recruiter interest prediction

### ⚡ One Click Apply AI
- Auto-tailors resume
- Generates cover letter
- Simulates real application process
- Shows:
  - 🤖 Applied by AI
  - Application history

### 🎤 AI Mock Interview System
- Dynamic AI-generated questions
- Voice + Text answering support
- AI follow-up questions
- Interview analytics:
  - Confidence score
  - Keyword usage
  - Weak areas

### 📊 Career Analytics Dashboard
- Skill demand trends
- Rejection insights
- Skill decay warnings
- Weekly performance report

### 📋 Kanban Job Tracker
- Track job status:
  - Applied → Interviewing → Offer → Rejected
- Visual job pipeline

### 🧠 AI Career Assistant (Chatbot)
- Ask anything about:
  - Jobs
  - Resume
  - Salary improvement
  - Career growth

### 🧪 Internship + Job Modes
- Toggle between jobs and internships
- Separate listings for each

### 🔍 Job Comparison Feature
- Compare multiple jobs side-by-side
- Helps make better decisions
